## User Stories
```
As a bank client
I want to make a deposit of 1000 on 10-01-2012 and a deposit of 2000 on 13-01-2012
```

```
As a bank client
I want to make a withdrawal of 500 on 14-01-2012
```

```
As a bank client
To see the history of my transactions
```
